# IssuesApp (issues-app)

## Install the dependencies
```bash
yarn
# or
npm install
```


## Configure ENV variable
```bash
GITHUB_TOKEN=
GITHUB_REPO=react
GITHUB_USER=facebook
```


### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```


### Lint the files
```bash
yarn lint
# or
npm run lint
```


### Format the files
```bash
yarn format
# or
npm run format
```



### Build the app for production
```bash
quasar build
```
